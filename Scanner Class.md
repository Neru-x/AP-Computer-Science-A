# **Scanner Class**
- grabs whatever is inputted in the console
- use "System.in"

## Example
```javascript
Scanner keyboard = new Scanner(System.in);
int num1, num2;
double dnum1, dnum2;

System.out.print("Eter an integer: ");
num 1 = keyboard.nextInt();
System.out.print("Enter a double value: ");
dnum1 = keyboard.nextDouble();
```
